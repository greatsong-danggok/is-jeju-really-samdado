## 1. 우리가 발견한 문제 (문제 발견 및 정의)

제주도는 예로부터 **'삼다도(三多島)'**, 즉 돌·바람·여자가 많은 섬이라고 불려왔습니다. 돌과 바람은 화산섬이라는 지형에서 쉽게 납득이 됩니다. 그런데 **'여자가 많다'는 말도 오늘날까지 사실일까요?**

우리 팀은 이 익숙한 표현이 **지금의 데이터로도 뒷받침되는지** 궁금했습니다. 그래서 막연한 통념을 다음과 같이 **검증 가능한 질문**으로 바꿨습니다.

> **"2026년 현재, 제주도의 여성 비율은 전국 17개 시도 중 통계적으로 상위권에 속하는가?"**

세부 탐구 질문:
1. 제주도의 여성 비율은 전국에서 몇 위인가?
2. 어느 연령대에서 여성이 가장 많은가?
3. 제주 내부(제주시·서귀포시·읍면동)에서는 어떻게 다른가?

> 📄 이 분석은 **2개 이상의 페이지(섹션)**로 구성되어 있습니다: 표지·연구동기 / 가설 설정 / 검증①②③ / 가설 재해석 / 결론 / 원자료 탐색.

## 2. 사용한 데이터 (데이터 리터러시)

| 항목 | 내용 |
|------|------|
| 출처 | 행정안전부 주민등록 인구통계 (https://jumin.mois.go.kr) |
| 데이터 | 연령별(0~100세+) 남녀 인구현황 |
| 기준 | 2026년 4월 |
| 분석 단위 | 17개 시도 + 제주 시군구 2곳 + 제주 읍·면·동 |

> ⚠️ 원본 CSV는 `cp949` 인코딩에 숫자에 천 단위 쉼표가 섞여 있어, 코드에서 인코딩 지정·행정구역 코드 분리·시도/시군구/읍면동 단위 구분 등의 전처리를 거쳤습니다.

**지표 정의**
- 여성 비율 = 여자 ÷ 전체 인구 × 100
- 성비 = 남자 ÷ 여자 × 100 (여자 100명당 남자 수)

## 3. 분석 방법

세 가지 관점에서 교차 검증했습니다.

1. **시도별 여성 비율 비교** — 제주가 전국 17개 시도 중 몇 위인지 확인
2. **연령대별 성비 곡선 + 인구 피라미드** — '여자가 많다'가 특정 연령대에서라도 사실인지 확인
3. **제주 내부 비교** — 제주시·서귀포시, 그리고 읍면동 단위로 내부 편차 확인

## 5. 우리가 내린 결론

(분석 결과에 맞게 본인 팀의 결론을 적으세요. 예시 ↓)

> 2026년 4월 인구 데이터에서 제주도의 여성 비율은 거의 정확히 남녀 1:1 수준으로, 17개 시도 중 상위권이 아니었습니다. 여성 우세는 고령층에서만 나타나는 **전국 공통 현상**이었습니다. 따라서 **인구 데이터 관점에서 '삼다도(여자가 많다)' 가설은 기각**됩니다. 다만 '삼다도'는 통계적 사실이 아니라 제주의 역사·노동·풍경을 응축한 **문화적 표현**으로 이해할 수 있습니다.

## 6. AI를 어떻게 활용했나 (AI 리터러시)

이 프로젝트는 우리가 **스스로 문제를 정하고, AI를 도구로 활용**해 해결했습니다. AI가 대신 결정한 것이 아니라, **무엇을 물을지·어떤 결과를 받아들일지는 우리가 판단**했습니다.

- 우리가 정한 분석 설계(시도 비교 → 연령대 → 내부 비교)에 따라 AI에게 코드 작성을 요청했습니다.
- AI의 답을 그대로 붙여넣지 않고, 우리가 의도한 분석인지 확인한 뒤 수정했습니다.
- 막힐 때마다 질문을 구체적으로 다시 던지며 단계적으로 완성했습니다.

> 💡 **AI 제안, 인간 결정** — AI는 거들었고, 문제 정의와 해석은 우리가 했습니다.

📎 **AI와 나눈 전체 채팅 기록을 함께 제출합니다.** (캡처 또는 공유 링크)

## 7. AI의 오류를 어떻게 검증했나 (비판적 사고력)

AI는 종종 그럴듯하지만 틀린 답(환각, hallucination)을 줍니다. 우리는 AI의 결과를 **무비판적으로 믿지 않고 직접 확인**했습니다. (아래는 작성 예시이며, 본인 팀이 실제 겪은 사례로 바꾸세요.)

- AI가 처음 작성한 코드가 **여성 비율과 성비 계산식을 헷갈려** 값이 이상하게 나와, 정의를 직접 확인하고 바로잡았습니다.
- AI가 제시한 통계 해석 중 "고령층 여성이 많은 것이 제주만의 특징"이라는 설명이 의심스러워, **다른 시도 데이터와 비교**해 전국 공통 현상임을 확인하고 결론을 수정했습니다.
- 0으로 나누는 오류(특정 연령에 여성 인구가 0인 경우)를 발견해 예외 처리를 추가했습니다.

> 데이터와 AI의 답을 **대조하며 검증하는 것**이 비판적 사고의 핵심이라고 생각합니다.
