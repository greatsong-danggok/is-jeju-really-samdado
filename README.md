# 🌋 제주도가 ‘삼다도’라는 말은 진짜인가?

행정안전부 주민등록 인구통계(2026.04)를 바탕으로,
‘제주=여자가 많은 섬(삼다도)’이라는 통념을 데이터로 검증하는 **탐구 보고서 대시보드**.

## 📁 구조

```
jeju_samdado_dashboard/
├── app.py                            # Streamlit 메인 앱
├── requirements.txt                  # 의존성
├── data/
│   └── 202604_연령별인구현황_월간.csv  # 행정안전부 원자료
└── README.md
```

## 🚀 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ Streamlit Community Cloud 배포

1. 이 폴더 전체를 GitHub 리포지토리로 푸시합니다.
   (CSV 파일을 함께 커밋해야 합니다 — 약 3.4MB라 LFS 없이도 가능)
2. https://share.streamlit.io 에 접속해 GitHub 계정으로 로그인.
3. **New app** → 리포지토리 선택 → Main file: `app.py` → **Deploy**.
4. 약 1~2분 뒤 공개 URL이 생성됩니다.

## 🧭 보고서 구성 (사이드바 목차)

1. **표지 · 연구 동기** — 가설의 출발점과 핵심 지표 미리보기

4. **검증 ② 연령대별 성비** — 다중 시도 비교 라인차트 + 제주 인구 피라미드
5. **검증 ③ 제주 내부** — 제주시·서귀포시·읍면동 TOP/BOTTOM 10

7. **결론 및 제언** — 최종 판정과 후속 탐구 아이디어
